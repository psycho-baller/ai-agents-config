# Report Contract

Per-file markdown reports always write to `analysis/<same-name>.md` beside the source markdown file.

Per-file JSON reports always write to `analysis/<same-name>.json` beside the source markdown file.

The JSON contract contains:
- `source`
- `summary`
- `findings`
- `vocabulary`
- `sentence_upgrades`
- `strengths`
- `practice_systems`
- `activation_loop`
- `evidence`
- `metadata`
