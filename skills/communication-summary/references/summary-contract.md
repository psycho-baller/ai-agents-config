# Summary Contract

The summary skill writes:
- `analysis/index.md`
- `analysis/index.json`

The summary JSON uses the same top-level contract as the per-file analysis artifacts:
- `source`
- `summary`
- `findings`
- `vocabulary`
- `strengths`
- `practice_systems`
- `activation_loop`
- `evidence`
- `metadata`

`metadata.file_priorities` is the authoritative ordered list of which file reports to open first.
